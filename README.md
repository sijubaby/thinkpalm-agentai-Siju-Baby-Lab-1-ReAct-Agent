## Lab Name
Minimal Python ReAct Agent – Kerala Restaurant Finder Tool

## Description
This project is a minimal Python ReAct (Reason + Act) Agent developed in Google Colab.

The application accepts a Kerala district name as input and identifies a popular restaurant from the predefined dataset.

The ReAct Agent demonstrates:
- Thought process
- Action execution
- Observation handling
- Final answer generation

## Features
- User input for Kerala district
- Restaurant recommendation
- Simple ReAct workflow
- Beginner-friendly implementation
- Google Colab compatible

## Technologies Used
- Python
- Google Colab

## How It Works
1. User enters a Kerala district name.
2. The ReAct agent identifies the district.
3. The tool searches the restaurant dataset.
4. The result is displayed as:
   - Thought
   - Action
   - Observation
   - Final Answer

## Sample Input
Thrissur

## Sample Output
A popular restaurant in Kozhikode is Paragon Restaurant.
